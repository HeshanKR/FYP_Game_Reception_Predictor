# Dataset

**Source:** Steam Games Dataset 2025 by artermiloff  
**URL:** https://www.kaggle.com/datasets/artermiloff/steam-games-dataset  
**File to download:** `games_march2025_cleaned.csv`  
**Place it at:** `data/raw/games_march2025_cleaned.csv`

The dataset is not committed to this repository due to file size.
After downloading, the notebooks and src modules reference it from `data/raw/`.
